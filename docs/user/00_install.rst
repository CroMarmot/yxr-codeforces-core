Install
=======

With github
-----------

.. code-block::

    git clone git@github.com:CroMarmot/yxr-codeforces-core.git
    cd yxr-codeforces-core
    # Install at local
    pip3 install -e . --config-settings editable_mode=compat

With pypi
---------

.. code-block::

    pip install yxr-codeforces-core
